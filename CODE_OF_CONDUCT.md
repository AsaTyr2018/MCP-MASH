# Code of Conduct

MCP-MASH is a security- and privacy-sensitive project. We expect project discussions to stay respectful, factual, and focused on improving the software.

## Expected Behavior

- Be respectful and constructive.
- Keep reports focused on reproducible behavior and concrete risk.
- Redact tokens, private mail data, generated private scripts, and run logs before posting.
- Assume good intent while still being precise about security boundaries.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Publishing secrets, tokens, private mail data, or exploit details in public issues.
- Repeated off-topic or bad-faith participation.

## Enforcement

Maintainers may edit, hide, or remove content and may restrict participation when needed to protect users or the project.

Security vulnerabilities should be reported privately through the process described in `SECURITY.md`.
